# reusable_iap Example

This example keeps the UI in the app layer while using `reusable_iap` as a headless service.

Before running it:

- Replace the sample product IDs in `example/lib/main.dart` with your own store product IDs.
- Configure in-app purchases in App Store Connect / Google Play Console.
- Run the example on a real device or emulator configured for store testing.
- Replace the sample `verifyPurchase` callback with your production verification flow when needed.
